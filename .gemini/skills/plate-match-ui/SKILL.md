# PlateMatch UI & UX

## Design Principles
- **Aesthetic**: Dark-mode primary (`bg-neutral-900` or `bg-black`), neutral backgrounds behind images, high-contrast action buttons.
- **Focus**: Minimum cognitive load. The reviewer's focus should be on the image.
- **Responsiveness**: Three-column desktop (1280px+), two-column tablet, single-column mobile (Section 6.8).

## Layout Components
- **Top Nav**: Progress summary + Primary Export button (Green).
- **Image Viewer**: Maximum available space, preserved aspect ratio, loading skeletons.
- **Side Panel**: Metadata (Filename, Status, Badge) + Control Group.
