# PlateMatch Project Structure

## specs/ Directory
- **README.md**: Central tracking for all phases.
- **phase-X/plan.md**: Strategic overview of the phase.
- **phase-X/task.md**: Granular tasks for tracking (T001, T002, etc.).
- **phase-X/implementation.md**: Technical specs and architecture.

## Implementation Workflow
1. Read `plan.md`, `task.md`, and `implementation.md` for the active phase.
2. Execute tasks sequentially.
3. Update `task.md` with `[x]` for completed items.
4. Update `specs/README.md` progress percentage.
